# Pentazein

B2B landing page for Pentazein, a garment manufacturing company based in Pakistan with management offices in Canada and Pakistan.

## Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

## Getting Started

```sh
npm install
npm run dev
```

Dev server runs at http://localhost:8080.

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server |
| `npm run build` | TypeScript compile + Vite build |
| `npm run preview` | Preview production build |
| `npm run lint` | Run ESLint |
| `npm run test` | Run all tests |

## Deployment

Deployed to GitHub Pages. Production build outputs to `dist/`.
